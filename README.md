# 🛡️ Hybrid Anomaly and Attack Classification Using Isolation Forest and Random Forest

## 📌 Overview

This project implements a Hybrid Intrusion Detection System (IDS) using Machine Learning techniques to detect anomalous network activities and classify cyber attacks. The system combines the strengths of Isolation Forest for anomaly detection and Random Forest for attack classification, providing an effective approach to network security monitoring.

## 🚀 Features

* 🔍 Detection of anomalous network behavior using Isolation Forest
* 🎯 Classification of cyber attacks using Random Forest
* 🧹 Data preprocessing and normalization
* 📊 Feature selection for improved model performance
* 🤖 Supervised and unsupervised learning techniques
* 📈 Model training, testing, and evaluation

## 🛠️ Technologies Used

## ⚙️ Workflow

1. 📥 Data Collection

5. 🛡️ Attack Classification using Random Forest
6. 📊 Model Evaluation
7. 📈 Result Analysis

## 🌲 Machine Learning Algorithms

### 🔍 Isolation Forest

An unsupervised learning algorithm used to identify anomalous network activities by isolating unusual observations.

### 🌳 Random Forest

A supervised learning algorithm used to classify detected anomalies into different attack categories.

## 📊 Results

The hybrid approach improves intrusion detection by combining anomaly detection and attack classification techniques, enabling efficient identification of malicious network activities.
